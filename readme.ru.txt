********************
** Общие сведения **
********************

DataExpress Web Server - веб-сервер, позволяющий работать с базами данных DataExpress через браузер с любого устройства. Веб-сервер может обрабатывать одновременно множество соединений к множеству баз данных. Не требует графического окружения, поэтому может работать на VDS.

Для корректной работы на устройствах требуется современный браузер на основе Chromium (Google Chrome, Yandex, Microsoft Edge и т. д.). Веб-интерфейс адаптируется под экран устройства, что обеспечивает удобную работу как на настольном компьютере, так и на смартфоне.

В настоящее время доступны альфа-версии для Windows 32-bit и Linux 64-bit, которые можно использовать в том числе в коммерческих целях.

Веб-сервер поддерживает два режима запуска: как консольное приложение и как сервис. В первом случае достаточно просто запустить файл dxwebsrv. Для постоянной работы лучше запускать программу как сервис (будет рассказано далее).

Для более быстрой работы рекомендуется устанавливать веб-сервер на тот же компьютер (виртуальный сервер), где и база данных. В системе Windows даже необязательно устанавливать СУБД Firebird 2.5, если планируется работать исключительно с веб-сервером. В Linux установка Firebird обязательна (статья по теме - http://dataexpress.wiki/stati:bd_v_oblake).

Более подробную информацию о веб-сервере вы найдете по ссылке http://dataexpress.wiki/web:web.

**********************************
** Установка и запуск в Windows **
**********************************

Программа будет работать во всех версиях Windows, начиная с Windows XP. Достаточно просто распаковать архив с программой в любую папку. Для запуска в режиме консольного приложения просто запустите файл dxwebsrv.exe. Откройте браузер и наберите в адресной строке: 
http://127.0.0.1/demodb/
Должно появиться окно ввода имени пользователя и пароля. Ничего не вводите, просто нажмите кнопку "Вход".

Для постоянной работы рекомендуется запускать веб-сервер как сервис. Для этого запустите командную строку от имени администратора, перейдите в папку программы и запустите install.bat. Например:
d:
cd d:\dxwebsrv_win32\
install.bat

Для удаления сервиса в командной строке от имени администратора запустите uninstall.bat.

*********************************************
** Установка и запуск в Ubuntu Linux 16.04 **
*********************************************

Сервер тестировался только в Ubuntu Linux 16.04, но скорее всего будет работать и в других сборках. Достаточно просто распаковать архив с программой в любую папку. Для корректной работы веб-сервера с демо-базой сделайте пользователя firebird владельцем файла DEMO_DB.FDB:
cd /home/dxwebsrv_linux64/
chown firebird DEMO_DB.FDB

Наберите в консоли ./dxwebsrv, чтобы запустить веб-сервер, как консольное приложение. Убедитесь, что у файла есть права на запуск. 

Запуск демона
------------------
1. Создайте или скопируйте в папку /etc/systemd/system файл dxwebapp.service. Готовый файл присутствует в дистрибутиве программы. Файл должен быть со следующим содержимым:

[Unit]
Description=DataExpress Web Server
After=network.target

[Service]
Type=simple
ExecStart=/var/web/dxwebapp -r
RemainAfterExit=yes
TimeoutSec=25

[Install]
WantedBy=multi-user.target

В параметре ExecStart введите действительный путь к файлу программы.

2. Выполните следующие команды для запуска демона:
systemctl enable dxwebapp.service
systemctl start dxwebapp.service

3. Готово. Проверить статус демона можно командной:
systemctl status dxwebapp.service

Проблема с локалью
-----------------------
Сервер может выдавать ошибки в фильтрах, выражения, раскраска не работают... Это все из-за локали. Нужно установить русскую локаль. Для этого выполняем следующие команды (с правами администратора):
sudo apt install language-pack-ru
sudo dpkg-reconfigure locales
Выбираем локаль ru_RU.UTF-8 UTF-8 и выбираем ее по умолчанию.
Ссылка на статью: https://obu4alka.ru/rusificziruem-ubuntu-server-16-04-lts.html
